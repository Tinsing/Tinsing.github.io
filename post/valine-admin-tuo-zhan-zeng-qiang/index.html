Valine Admin拓展增强

当你的个人博客/独立网站接入Valine后，你会发现，怎么没有评论数据管理呢?😅

引用Valine官网的描述

由于Valine 是无后端评论系统，所以也就没有开发评论数据管理功能。请自行登录Leancloud应用管理。

那么我们应该怎么做才能优雅的使用评论管理功能呢？😎

细心看过Valine官网的同学就可能会知道了，Valine作者有给出一个解决的好办法，它就是——— Valine-Admin 。

引用一下Valine Admin项目的描述

Valine Admin 是 Valine 评论系统的扩展和增强，主要实现评论邮件通知、评论管理、垃圾评论过滤等功能。支持完全自定义的邮件通知模板。基于Akismet API实现准确的垃圾评论过滤。此外，使用云函数等技术解决了免费版云引擎休眠问题，支持云引擎自动唤醒，漏发邮件自动补发。兼容云淡风轻及Deserts维护的多版本Valine。

温馨提示:使用国际节点的同学在设置定时任务时需注意时区问题。例如发邮件的Cron表达式可以改为0 0 0 * * ?；半小时后唤醒可以改为0 */30 0-15 * * ?。

在配置过程中如果遇到问题欢迎在评论区留言或去相关教程链接请教作者。😄🥂（这个小站也是接入Valine评论平台的，简洁高效）
